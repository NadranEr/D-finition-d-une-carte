## Format et publication
- Essai académique rigoureux, accessible sans être simplifié
- 15–20 pages (cible : ~18 pages)
- Publication : indéfinie pour l'instant — on attend le résultat final

- Archive LaTeX via Overleaf + GitHub
## Structure du projet
- Repo GitHub connecté à Overleaf
- Fichiers `.tex` séparés par section + `main.tex` qui compile
- Fichier `bibliography.bib` pour les sources (BibTeX)
- Contexte Claude : `conversation_cartes.md`, `plan.md`, `decisions.md`, `project_instructions.md`
## Style
- Prose narrative, pas de listes à bullets dans le corps du texte
- Première personne dans le prologue
- Les sources citées naturellement dans le texte, pas uniquement en note
- Ton académique mais pas jargonneux
## Posture éditoriale
- Transparence sur l'usage de l'IA comme interlocuteur dialectique (mentionné dans le prologue)
- Regard critique maintenu sur les réponses de l'IA tout au long de la conversation source
- La contribution de l'IA à la synthèse bibliographique est reconnue

## Thèse
La carte est une **dyade** entre :
- Un créateur qui réduit intentionnellement un espace à une métrique globale cohérente
- Un lecteur qui s'y projette librement sans position imposée
La cartéité est graduée et relationnelle — pas binaire.
 
## Contribution par rapport à la littérature
Lien causal entre métrique globale, habitabilité et ouverture interprétative — non formulé ainsi dans la littérature existante consultée.
